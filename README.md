# video downloader
Simple Link Video Downloader
This project provides a simple command-line tool written in Python to download videos from  websites. It leverages the powerful pytube library to fetch videos with ease.

Features:
Download  videos with direct link.
Easy-to-use interface.

Run the .exe file for easy use of the program.
